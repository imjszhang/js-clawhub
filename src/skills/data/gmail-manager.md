# Gmail Manager

智能邮件管理，让 AI 助手帮你处理收件箱。

## 功能特性

- **邮件摘要**：每日自动总结重要邮件
- **智能回复**：根据上下文自动起草回复
- **垃圾退订**：批量退订不需要的订阅邮件
- **分类整理**：自动将邮件归类到对应标签
- **重要提醒**：识别紧急邮件并主动通知

## 配置步骤

1. 在 Google Cloud Console 创建 OAuth 凭据
2. 配置 Gmail API 权限
3. 在 OpenClaw 中添加 Gmail 技能

```bash
openclaw skill install gmail-manager
```

## 使用示例

- "帮我看看今天有什么重要邮件"
- "退订所有营销邮件"
- "帮我回复 John 的邮件，告诉他周五可以"
- "每天早上 8 点给我发邮件摘要"

## 安全说明

- OAuth 2.0 标准授权，不存储密码
- 所有邮件数据仅在本地处理
- 可随时在 Google 账户中撤销权限
