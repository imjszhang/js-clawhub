# Welcome to JS ClawHub

JS ClawHub 是一个由 JS 策展的 OpenClaw 生态项目导航站。

## 为什么做这个站？

OpenClaw 的生态发展速度惊人。短短几周内，社区已经创造了大量技能、集成方案和使用案例。但这些信息散落在 Twitter、Discord、GitHub 各处，新用户很难快速找到自己需要的东西。

ClawHub 就是为了解决这个问题。

## 这个站点收录什么？

- **项目导航**：OpenClaw 生态中的各类项目，按分类整理
- **技能市场**：社区创建的各种技能，附带安装说明
- **博客教程**：使用教程、评测文章、技术分享
- **入门指南**：从零开始配置 OpenClaw 的完整指南

## JS 的策展视角

这不是一个中性的项目目录。每一个收录的项目，都经过了我（JS）的亲自体验和筛选。精选推荐里包含我的使用感受和评价。

如果你有好的 OpenClaw 项目想要推荐，欢迎通过 GitHub Issues 提交。

## 技术栈

这个站点本身也是一个有趣的技术实践：

- 纯静态站点，部署在 GitHub Pages
- Neo-Brutalism 设计风格（JS 品牌视觉系统）
- 数据驱动：所有内容通过 JSON + Markdown 管理
- 零构建部署：push 即发布

欢迎探索，希望你能在这里找到有用的东西。
