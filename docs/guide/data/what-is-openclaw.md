# OpenClaw 是什么

OpenClaw 是一个开源的个人 AI 助手，由 Peter Steinberger (@steipete) 创建。它运行在你自己的电脑上，通过你已有的聊天应用（WhatsApp、Telegram、Discord 等）与你交互。

## 核心特点

### 运行在你的机器上

OpenClaw 不是云端服务，它运行在你的 Mac、Windows 或 Linux 机器上。你的数据完全私有，不会上传到任何第三方服务器。

### 持久记忆

不同于普通的聊天机器人，OpenClaw 拥有持久记忆。它会记住你的偏好、你的习惯，随着使用变得越来越了解你。

### 通过聊天应用交互

你不需要学习新的界面。直接在 WhatsApp、Telegram、Discord 等你已有的聊天应用中发消息，就像和朋友聊天一样。

### 技能系统

OpenClaw 的能力可以通过技能（Skills）无限扩展。社区已经创建了大量技能，你也可以自己创建，甚至让 AI 自己编写新技能。

### 完整系统访问

OpenClaw 可以：

- 浏览网页、填写表单
- 读写文件、运行脚本
- 管理邮件、日历
- 控制智能家居设备
- 几乎所有你能在电脑上做的事

## 它不是什么

- **不是另一个聊天机器人**：它能执行真实操作，不只是对话
- **不是云端服务**：你拥有完全控制权
- **不是封闭生态**：开源、可 hack、社区驱动
