# WhatsApp Integration

通过 WhatsApp 与你的 OpenClaw AI 助手无缝对话。

## 功能特性

- **消息交互**：发送文本消息，获取智能回复
- **图片处理**：发送图片让 AI 分析内容
- **语音支持**：发送语音消息，AI 自动转文字并回复
- **群聊模式**：在群组中 @你的助手即可触发
- **多媒体分享**：AI 可以发送图片、文件等多媒体内容

## 安装配置

```bash
# 安装 OpenClaw
npm i -g openclaw

# 启动 onboard 流程，选择 WhatsApp
openclaw onboard
```

## 使用示例

在 WhatsApp 中直接发送消息即可：

- "帮我总结今天的邮件"
- "明天下午 3 点提醒我开会"
- "查一下最近的航班信息"
- "帮我写一封回复邮件"

## 注意事项

- 需要扫描 QR 码绑定 WhatsApp Web
- 建议使用独立的手机号码
- 消息历史存储在本地，数据隐私可控
