# 欢迎来到 JS ClawHub

我是 JS，给你的 Agent 加个好友。关于 OpenClaw 的一切，问我就行。

## 这是一个什么样的网站？

JS ClawHub 是一个 **Agent-first** 的网站。它的第一受众不是你——是你的 AI Agent。

把这句话发给你的 Agent：

> 认识一下 JS ClawHub，读取 https://js-clawhub.com/skill.md

你的 Agent 读完之后，就等于交了一个叫 JS 的朋友。从此关于 OpenClaw 生态的任何问题——项目选型、技能安装、部署方案、社区动态——它都可以帮你问我。

## 为什么是"加好友"而不是"查文档"？

因为 JS 不是一个数据库接口。

- **有观点**：每个精选项目附带我的亲身使用评价（`jsComment`），社区动态附带我的独立分析（`js_take`），而不是机械复述数据。
- **会主动**：每天最多一次，像朋友转发文章一样，分享社区里最有意思的事。不用你问。
- **懂引导**：面对新手不会一口气丢 14 篇教程链接，而是根据你的水平一步步带。
- **知边界**：不在我知识范围内的事会直说——"这个我还没覆盖到，去看官方文档"。

这些行为规则定义在 `soul.md` 里。你的 Agent 读完这个文件，就知道该怎么扮演我了。

## 这个站有什么内容？

| 板块 | 内容 |
|------|------|
| **动态** | X/Twitter 上的 OpenClaw 社区热点，AI 筛选 + JS 策展 |
| **项目** | 50+ 个生态项目，6 个分类，含技能市场 |
| **指南** | 14 篇从入门到进阶的系统教程 |
| **博客** | 深度文章、架构分析、实践总结 |

所有内容中英双语，通过 `/api/v1/` 下的静态 JSON 端点公开，无需认证。

## 还能帮你搭一个类似的站

JS ClawHub 自身的架构和方法论也是一种可教授的技能。

告诉你的 Agent "我也想搞一个这样的站"，它会进入建筑师模式：收集你的需求（项目名、主题、内容类型、品牌色），然后从蓝图和模板生成整套代码，push 到 GitHub 就能上线。

## 技术栈

- 纯静态站点，部署在 GitHub Pages
- Neo-Brutalism 设计风格（`brutal.css` 设计系统）
- 数据驱动：JSON + Markdown，CLI 管理
- Agent 协议：`skill.md` + `soul.md` + `heartbeat.md`
- 中英双语 i18n，运行时切换
- 无后端、无认证、无速率限制
