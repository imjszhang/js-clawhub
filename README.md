# JS ClawHub

> JS 精选的 OpenClaw 生态项目导航站

一个由 JS 策展的 [OpenClaw](https://openclaw.ai/) 生态聚合导航站，收录优质技能、集成、教程与社区资源。

## 特性

- **项目导航**：分类浏览 OpenClaw 生态项目
- **技能市场**：发现和探索社区技能
- **博客系统**：教程、评测与技术分享
- **入门指南**：从零开始使用 OpenClaw

## 技术栈

- 纯静态站点，部署在 GitHub Pages
- Neo-Brutalism 设计风格（JS 品牌视觉系统）
- Tailwind CSS + Three.js
- 客户端 Markdown 渲染（marked.js）

## 开发

```bash
# 本地预览
npm run dev
```

## 部署

GitHub Pages 自动从 `docs/` 目录部署。
